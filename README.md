# Transaction-Tracker-API
An API for the Transaction Tracker app, developed using Node.js.


### Database Setup
- Navigate to the `database` folder.
- Execute `db.sql` to create the database.
- Execute `tables.sql` to create the required tables.


### API Setup
- Navigate to the `api` folder.
- Install all the required packages using `npm i`.
- Create `.env` file and mention the following information:
    ```
    PORT=
    DB_HOST=
    DB_USERNAME=
    DB_PASSWORD=
    DB_NAME=
    ```
- Start the server using `node app`.
